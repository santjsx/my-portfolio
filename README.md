# Santhosh Reddy — Portfolio

A high-performance, cinematic developer portfolio built with Neo-Minimalist aesthetics and butter-smooth motion.

## 🛠️ Tech Stack
- **Engine**: Vanilla HTML, CSS, JS
- **Motion**: GSAP + Lenis Scroll
- **Real-time**: Lanyard API (Discord Integration)

## 📡 Dynamic Controls (Lanyard)
You can update your site live via Discord DMs to the **Lanyard bot**:

| Key | Command | Action |
|---|---|---|
| `mood` | `.set mood <text>` | Updates the bridge/status text |
| `hero_quote` | `.set hero_quote <text>` | Changes the hero section quote |
| `accent_color` | `.set accent_color <hex>` | Real-time global theme change |
| `about_photo_url` | `.set about_photo_url <url>` | Swaps the profile portrait |
| `skills` | `.set skills Name:Level` | Appends new technical credits |

## 🚀 Setup
```bash
git clone https://github.com/santjsx/my-portfolio.git
npx serve .
```
