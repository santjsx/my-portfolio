# Neo-Minimalist Portfolio — Santhosh Reddy

A high-performance, ultra-premium developer portfolio designed with a focus on **Neo-Minimalism**, typographic tension, and high-quality micro-interactions.

## Architecture & Aesthetic

This portfolio embodies the "Neo-Minimalist" philosophy: clean lines, generous negative space, and architectural typography. It leverages a sophisticated palette of **Off-White**, **Charcoal**, and a subtle **Sage** accent, with seamless transitions between light and dark modes.

## Technology Stack

### Frontend Core
- **HTML5**: Semantic structure.
- **Vanilla CSS**: Custom design system using CSS Variables and modern Layout (Grid/Flexbox).
- **Modern JavaScript**: Core interactive logic.

### Experience & Animation
- **GSAP (GreenSock)**: Orchestrates advanced entrance animations and scroll-triggered transitions.
- **Lenis**: Provides butter-smooth inertial scrolling for a cinematic digital experience.
- **Remix Icons**: High-quality vector iconography.

## Key Sections

1. **Hero**: A massive typographic entrance featuring ambient gradients and noise textures.
2. **About**: A refined editorial layout with parallax image reveals.
3. **Expertise**: A dense "Technical Index" with interactive hover states.
4. **Selected Work**: A compact, agency-style index of projects focusing on density and visual hierarchy.
5. **Contact**: A massive typographic "Let's Talk" interaction with a real-time local clock and status matrix.

## Development

The project is built to be extremely lightweight and serves from a single-page structure.

```bash
# Clone the repository
git clone https://github.com/santjsx/my-gtaV-themed-portfolio.git

# Serve locally
npx serve .
```
